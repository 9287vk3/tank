# 🚀 Танк в лабиринте: Дуэль с тенями

Классическая аркадная игра, где вы управляете танком, двигаетесь по запутанному лабиринту, уничтожаете движущихся врагов и стараетесь выжить. Не дайте себя догнать!

![Gameplay preview](https://via.placeholder.com/800x400?text=Танк+в+лабиринте)

## 🎮 Игровой процесс

- Вы — зелёный танк в центре событий.
- По лабиринту бродят **красные агрессивные сферы** (враги). Они движутся к вам, стараясь столкнуться.
- Ваша задача — расстреливать их из пушки, не допуская касания.
- У вас **3 жизни**. При столкновении жизнь теряется, враг отбрасывается, а танк получает кратковременную неуязвимость.
- За каждого уничтоженного врага начисляются очки. Чем больше очков — тем больше врагов появляется одновременно (до 8).

## 🕹️ Управление

| Клавиша               | Действие                         |
|-----------------------|----------------------------------|
| **Стрелки (↑ ↓ ← →)** | Движение танка по лабиринту      |
| **Пробел (Space)**    | Выстрел                          |

> ⚡ Совет: ведите огонь на упреждение — враги двигаются быстро и постоянно меняют траекторию.

## 📦 Как запустить

### 1. Локальный запуск (проще всего)
- Сохраните весь код игры в файл `index.html`.
- Дважды кликните по файлу — игра откроется в вашем браузере.

### 2. Запуск с помощью локального сервера (рекомендуется)
```bash
# Если у вас установлен Python 3
python -m http.server 8080

# Или с помощью Node.js (live-server)
npx live-server
